Análise de Coerência Informacional (ICOER)

Ferramenta para calcular o Índice de Coerência Informacional diretamente no navegador. Cole seu texto abaixo e clique em Analisar.

Os resultados da análise aparecerão aqui.
from pyscript import Element def analyze_text(evt=None): # Recupera texto do textarea text = Element("text-input").element.value or "" # Exemplo de análise: conta palavras score = len(text.split()) result_text = f"Índice de Coerência Informacional (ICOER): {score}" # Atualiza o div de resultado Element("result").element.innerText = result_text # Vincula o evento de clique ao botão Element("analyze-btn").element.addEventListener("click", analyze_text)
