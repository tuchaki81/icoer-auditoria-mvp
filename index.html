Análise de Coerência Informacional (ICOER)

Ferramenta para calcular o Índice de Coerência Informacional diretamente no navegador. Cole seu texto abaixo e clique em Analisar.

Os resultados da análise aparecerão aqui.
from pyscript import Element # placeholder: conte as palavras def analyze_text(evt=None): text = Element("text-input").element.value or "" score = len(text.split()) Element("result").element.innerText = f"ICOER (contagem de palavras): {score}" Element("analyze-btn").element.addEventListener("click", analyze_text)
